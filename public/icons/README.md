# PWA-Icons (Planenadler)

Ersetze die Platzhalter durch echte Icons:

- `icon-192.png` / `icon-512.png` – Standard-Icons (any)
- `icon-192-maskable.png` / `icon-512-maskable.png` – Maskable (mit Safe Zone)
- `apple-touch-icon.png` – z. B. 180×180 für iOS

Empfehlung: [PWA Asset Generator](https://www.pwabuilder.com/imageGenerator) oder eigenes 512×512-Design.
